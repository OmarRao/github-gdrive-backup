Google Drive

Authorise once with node src/auth/google-auth.js, then paste your Drive folder ID below.

Quick Setup Guide

  1. Copy .env.example → .env and fill in your tokens.
  2. Create OAuth credentials in Google Cloud Console and download google-client-secret.json into the credentials/ folder.
  3. Run node src/auth/google-auth.js and follow the browser prompt.
  4. Create a folder in Google Drive, copy its ID from the URL.
  5. Set GDRIVE_FOLDER_ID in your .env.
  6. Start a backup from the Backup tab — or trigger via GitHub Actions.
