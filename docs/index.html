Quick Setup Guide
  1. Clone the repo and run npm install
  2. Copy .env.example → .env and fill in your tokens
  3. Download OAuth credentials from Google Cloud Console → save as credentials/google-client-secret.json
  4. Run node src/auth/google-auth.js to authorise Google Drive
  5. Add all 5 secrets to your GitHub repository
  6. Trigger a backup from the Backup tab or wait for the daily schedule (02:00 UTC)
