Quick Setup Guide
  1. Google Cloud OAuth app — go to Google Cloud Console, create an OAuth 2.0 Client ID (Web application), add https://omarrao.github.io as an authorised JavaScript origin, and copy the Client ID into the Google Drive settings tab.
  2. Sign in with Google — click the Sign in with Google button in the Drive settings tab. This authenticates your browser for browsing Drive sessions.
  3. CLI auth for Actions — clone the repo locally, run node src/auth/google-auth.js, follow the browser prompt, then copy the contents of credentials/google-token.json into the GOOGLE_TOKEN Actions secret.
  4. Add all 5 secrets — go to the Actions Secrets tab and add each secret.
  5. Trigger a backup — go to the Backup tab, select repos, click Trigger.
